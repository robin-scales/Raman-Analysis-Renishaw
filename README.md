## Raman Analysis Renishaw
Developed by Dr Robin Scales, based on the work by Andreas Iakovis. 

Python version used by R Scales was 3.14.0

Enter the following commands into the cmd terminal in Windows (with the working directory set as the project folder):
> mkdir .venv
> 
> python3 -m venv .venv
> 
> .venv\Scripts\activate
> 
> pip install -r requirements.txt

You should now have an active virtual environment with the correct libraries installed.

To deactivate the virtual environment, run:
> deactivate
